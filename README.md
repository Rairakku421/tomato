# 🍅 Functions of this Tomato

### Task Management:
- Add your tasks
- Rank tasks by priority:
    - 🟢 Default (easy)
    - 🟡 So-so (medium)
    - 🔴 Important (hard)

### Pomodoro Timer:
1. Select a task
2. Start 25-minute timer
3. When the timer runs out, there is a 5-minute short break.
4. After 4 cycles 20-minute long break
